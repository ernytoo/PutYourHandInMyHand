# PutYourHandInMyHand
PutYourHandInMyHand is a mod for Gorilla Tag which allows you to hold other Monkes hands in Modded lobbies.

## What the mod does
- Adds Gorilla Climbables to all of the other Monkes hands so that you can hold onto them!
